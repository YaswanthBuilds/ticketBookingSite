# 🎟️ Ticket Booking Web Application

A full-stack web application that allows users to browse, select, and book tickets through an interactive and user-friendly interface. The system handles booking workflows, data storage, and user interactions efficiently.

---

## Features

* Browse available tickets/events
* Book tickets through a dynamic interface
* CRUD operations for ticket management
* Responsive UI for seamless user experience
* Backend integration for storing and retrieving booking data

---

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP *(update if different)*
* **Database:** MySQL

## Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/YaswanthBuilds/ticketBookingSite.git
   ```

2. Navigate to the project folder:

   ```bash
   cd ticketBookingSite
   ```

3. Set up your local server (XAMPP / WAMP / LAMP)

4. Import the database:

   * Open phpMyAdmin
   * Create a new database
   * Import the provided `.sql` file

5. Run the project:

   * Start Apache & MySQL
   * Open in browser:

     ```
     http://localhost/ticketBookingSite
     ```
